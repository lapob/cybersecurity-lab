# Argomenti in studio

Questa pagina contiene gli argomenti da studiare adesso, in ordine di priorità.

## Priorità 1 — Fondamenta solide

- Linux base
- Bash e comandi da terminale
- Git e GitHub
- Networking base
- HTTP e HTTPS

## Priorità 2 — Android Security

- Architettura Android
- Activity
- Service
- Broadcast Receiver
- Content Provider
- Intent e deep link
- Android Manifest
- Permessi Android
- ADB avanzato
- Logcat avanzato

## Priorità 3 — Analisi APK

- APKTool
- JADX
- Lettura del Manifest
- Analisi delle risorse
- Analisi del codice Java/Kotlin decompilato
- Ricerca di endpoint, chiavi, log sensibili e configurazioni deboli

## Priorità 4 — Mobile Pentesting Lab

- MobSF
- Burp Suite con dispositivo Android/emulatore
- Proxy HTTP/S
- Certificato CA in ambiente di test
- Frida base
- Objection base

## Metodo di studio

Per ogni argomento:

1. Capire la teoria.
2. Scrivere i comandi principali.
3. Fare un laboratorio semplice.
4. Salvare errori e soluzioni.
5. Creare una mini checklist.
