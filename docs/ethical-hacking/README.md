# Ethical Hacking

Questa sezione contiene il percorso per imparare ethical hacking in modo responsabile.

## Cos'è l'ethical hacking

L'ethical hacking è l'attività di analizzare sistemi, applicazioni e reti per trovare problemi di sicurezza prima che vengano sfruttati da attaccanti reali. La differenza fondamentale è l'autorizzazione.

## Competenze principali

- Linux
- Networking
- Web security
- Mobile security
- Analisi vulnerabilità
- Uso corretto dei tool
- Documentazione tecnica
- Reporting
- Comunicazione del rischio

## Mentalità corretta

Un ethical hacker non cerca solo di trovare problemi: deve capire impatto, contesto e soluzione.

Domande da farsi sempre:

1. Ho autorizzazione?
2. Qual è l'obiettivo del test?
3. Qual è il rischio di danneggiare qualcosa?
4. Come documento il risultato?
5. Come propongo una correzione?

## Percorso consigliato

1. Fondamenta Linux e rete
2. HTTP e web security
3. Android e mobile security
4. Analisi statica e dinamica
5. Laboratori controllati
6. Report professionali
7. Portfolio GitHub pulito
